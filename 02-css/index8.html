display: none

  • 요소를 화면에 표시하지 않음.
  • 요소가 차지하는 공간도 사라진다.
  • = 그냥 없애버리는 방법!

visibility: hidden

  • 요소만 사라지고 차지하던 공간은 그대로 남는다.
  • 모습과 속성을 숨긴다.

opacity: 0

  • 요소 투명도
  • 요소만 사라지고 차지하던 공간은 그대로 남는다.
  • 모습만 숨기고 속성은 남는다. (ex. 클릭, 호버, 포커스 등 이벤트 가능
  • 투명도 0 ~ 1 사이의 숫자로 조절 가능
